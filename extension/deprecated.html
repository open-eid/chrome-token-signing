Chrome Token Signing
Active development and management of the Token Signing component have ended due to the transition to the new web authentication and signing solution Web eID.
Token Signing extension will be removed from the extensions stores on 1st June 2023.
Web eID components are included in the ID-software installation package which can be downloaded from the website www.id.ee.
NB! Please remove Token Signing extension in order for Web eID to work.
For Uninstallation, please follow these instructions.
